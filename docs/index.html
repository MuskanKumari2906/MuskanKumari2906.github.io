Bedroom Anatomy VR

An interactive VR experience exploring the anatomy of a bedroom.

Overview

This project utilizes A-Frame to create an immersive virtual reality experience where users can interact with various objects in a bedroom setting.

Getting Started

To run the project locally, clone the repository and open src/index.html in a web browser that supports WebVR.

Features

  • Interactive 3D objects with information displays
  • Gaze-based and pointer controls for navigation
  • Responsive design for mobile and desktop users

Usage

Users can explore the virtual bedroom by looking at objects to learn more about them or clicking on them for additional information.
